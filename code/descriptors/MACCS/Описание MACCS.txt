С помощью открытой библиотеки RDkit возможно реализовать получение отпечатка MACCS из Smiles. При этом RDkit создает битовый отпечаток, используя 167 структурных паттернов, тогда как стандартная интерпретация MACCS ключей основана на 166-битной последовательности. RDKit сохраняет номера ключей MACCS, поэтому ставит нулевой бит в начале строки. Чтобы вычленить в стандартном виде MACCS Key достаточно просто проигнорировать первый бит.
В директории расположен код с функционалом по расчету MACCS, а также файл в формате excel с расшифровкой каждого бита в полученном 166-битном ключе. В ноутбуке представлен цикл, который позволяет получить понимание о значении каждого бита по каждому веществу из датасета.
